# Vector clock rules

- All elements are $\leq$, one element must be <: Happens before
- All elements are $\geq$, one element must be >: Happens after
- Otherwise: Concurrent

Lamport Clocks use up less bandwidth in a network than VCs, which could be significant.

- Also have to know number of total participants
