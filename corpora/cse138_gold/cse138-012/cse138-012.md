# Lecture #7

snapshots are correct when:

## Does C-L terminate?

- Assumptions
  - Fixed number of processes

### If every process:
- ① records its own process state 
- ② reads the states of all incoming channels
- C-L terminates.

- finished when every channel recieves a marker message

In C-L algo, at least one process has to initiate.

Multiple process initiation is actually a huge deal, becuz if this wasn't true, it would be bad. Processes would have to agree to decide which process gets to initiate.

- not good
- complexity

Decentralized algorithm: Can be independently initiated by different processes without them needing to coordinate with each other

Centralized algorithm: Must exactly have one initiator
