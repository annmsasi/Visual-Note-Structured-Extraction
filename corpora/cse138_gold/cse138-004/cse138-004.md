# CSE 138 Lecture 3

Fit by solving: $\min \text{Cost}(\theta)$

- Reflexive: Ordering relation includes itself
- Anti-symmetrical: Ordering relation is not reversible
- Transitivity: Ordering relations can be put together
- A real partial ordering has these 3 properties (not all elements are related).
- "happens before" is an irreflexive partial order

Totally ordered Set: All elements are related
- e.x: Natural numbers, ordered by the usual $\leq$ relation

A process order is totally ordered

## Logical Clocks (particularly, Lamport clocks)

- A way of assigning natural numbers to events
  - $LC(A) = 3 \leq LC = Lamport Clock$

Clock condition: If $a \rightarrow b$, then $LC(a) < LC(b)$

Lamport Clocks are consistent with the happens before relation. (also consistent with causal)
